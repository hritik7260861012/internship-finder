# 🚀 Internship Finder Platform

A full-stack web application to find and apply for internships.

---

## 📌 Features

- 🔐 User Registration & Login
- 📄 View Internships
- 📩 Apply to Internships
- 📊 Track Applications (Dashboard)
- 🔒 Authentication using userId (LocalStorage)

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Axios
- CSS (Custom UI)

### Backend:
- Spring Boot
- REST APIs
- Hibernate

### Database:
- MySQL
